# 🔍 SmartSearch - AI Powered Search Engine

A modern AI-powered Search Engine project with an advanced Admin Dashboard, Machine Learning integration, API-based intelligent search, and local fallback ranking system.

---

# 🚀 Features

## 🌐 User Search Engine Interface

* Modern and responsive UI
* Smart search suggestions
* AI-powered answers
* Search categories (Web, Images, Videos, News)
* Trending searches section
* Fast search results
* Mobile-friendly design

## 🛠️ Admin Dashboard

* Content Management System
* User Management
* Search History Tracking
* Database Monitoring
* API & ML Settings Panel
* Analytics Dashboard
* Real-time System Status

## 🤖 Machine Learning Features

* Tokenization

* Secure API Configuration

* Protected Dashboard Access

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/smartsearch.git
```

## 2️⃣ Open Project Folder

```bash
cd smartsearch
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
PORT=3000
```

## 5️⃣ Start Server

```bash
npm start
```
